class speechemotionrecognition.Model(save_path: str = '', name: str = 'Not Specified')[source]¶

Bases: object

Model is the abstract class which determines how a model should be. Any model inheriting this class should do the following.

  1. Set the model instance variable to the corresponding model class which which will provide methods fit and predict.
  2. Should implement the following abstract methods load_model, save_model train and evaluate. These methods provide the functionality to save the model to the disk, load the model from the disk and train the model and evaluate the model to return appropriate measure like accuracy, f1 score, etc.
